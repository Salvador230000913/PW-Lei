## Tema do Projeto
    • Website de Tracking de Voos em Tempo Real

**Ideia Geral** 
    • Desenvolver uma aplicação web interativa que permita visualizar voos em tempo real num mapa, utilizando dados reais de aviação.

    • A aplicação irá usar o Leaflet para renderizar o mapa e a OpenSky Network como fonte de dados para obter informações atualizadas dos voos.

    • O utilizador poderá acompanhar aeronaves, visualizar detalhes e interagir com os voos diretamente no mapa.

## Funcionalidades
    • Mapa interativo com suporte a zoom e navegação
    • Representação dos aviões com ícones dinâmicos
        Hover (passar o rato):
            Mostra informações básicas (callsign, altitude, velocidade)
        Clique no avião:
            Foca o mapa na aeronave
            Mostra informações detalhadas:
            País de origem
            Altitude
            Velocidade
            Direção (heading)
            Última atualização
        Atualização periódica da posição dos voos
        Trajetória do voo (linha no mapa)
    • Sistema de Favoritos
        Possibilidade de marcar voos como favoritos
        Lista de voos guardados pelo utilizador
        Destaque visual no mapa
    • Anotações
        Adicionar notas personalizadas a um voo
        Guardar observações (ex: “voo interessante”, “avião raro”)
    • Modo dark/light
